# Code of Conduct

Devworks Server OS is a technical project for learning, building, and testing a custom Debian-based server OS.

Participants are expected to:

- Be respectful and constructive.
- Keep discussions focused on the project.
- Report security issues privately or through the documented support path.
- Avoid sharing private keys, passwords, access tokens, or production secrets.
- Treat destructive installer behavior with care and clear warnings.

Unacceptable behavior includes harassment, personal attacks, spam, and intentionally misleading safety claims.

Project maintainers may remove disruptive comments, close unproductive issues, or block abusive accounts.
